# Simple Item Editor

Simple item editor is a minecraft mod for item editing using client commands.

## Features:

### `/edit` command:
- `... attribute ...`
- - `... get [<id>]`
- - `... set <id> <attribute> <amount> [<operation>] [<slot>]`
- - `... remove <id>`
- - `... clear`
- `... banner ...`
- - `... get [<index>]`
- - `... set <index> <pattern> <color>`
- - `... remove <index>`

- - `... base ...`
